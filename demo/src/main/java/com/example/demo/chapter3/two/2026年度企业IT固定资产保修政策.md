2026年度企业IT固定资产保修与报废政策
1. 资产生命周期管理
   为了优化企业运营成本，提升办公效率，IT资产管理部对2026年度新采购及在保的IT设备服务等级协议（SLA）进行了重新修订。对于过保且维修成本超过设备现值40%的设备，应直接走报废流程。
2. 设备保修期与响应SLA对照表
   以下条款适用于2026年1月1日之后入库的所有固定资产：
   设备类型/标准保修期/硬件故障响应SLA/是否提供备用
   机戴尔R740服务器/3年/4小时内上门/否
   联想ThinkPad T系列笔记本/2年/24小时内修复/是
   苹果MacBook Pro/1年/48小时内寄修/是
   戴尔/AOC 办公显示器/3年/3个工作日/否
   思科/华为 核心交换机/5年/2小时内上门/否
3. 笔记本电脑维修特别补充条款
   对于研发部门（研发中心编码：R&D-01至R&D-15）配备的联想或苹果笔记本电脑，若发生因意外进液或跌落导致的主板损坏（非自然损坏），不在标准免费保修范围内。此种情况需由该研发团队的成本中心（Cost Center）承担维修费用，IT部门仅负责提供备用机及数据恢复协助。
