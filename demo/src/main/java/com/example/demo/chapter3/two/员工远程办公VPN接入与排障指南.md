员工远程办公VPN接入与排障指南
概述
为了保障公司数据的安全性，所有在非办公网络环境下（如居家办公、出差）访问公司内网ERP、GitLab和核心财务系统的行为，均必须通过GlobalProtect VPN进行加密连接。

客户端下载与安装
Windows用户：请访问 https://vpn.company.internal/windows 下载最新的 6.1.2 版本。

macOS用户：请访问 https://vpn.company.internal/macos 下载，安装时需要在“系统偏好设置 -> 隐私与安全性”中允许网络扩展。

常见错误代码与解决方案
错误代码 401：凭证无效或过期
原因：AD域密码错误或密码已过90天强制修改期。
解决方案：请使用手机连接流量访问自服务密码重置平台 https://pwd.company.com 更改密码，等待5分钟同步后再试。

错误代码 502：网关无响应
原因：当前连接的边缘网关节点负载过高或所在运营商网络阻断了UDP 4501端口。
解决方案：在GlobalProtect客户端的“门户”设置中，将默认网关从 gw-cn-shanghai 手动修改为备用节点 gw-cn-hangzhou。

多因素认证（MFA）失败
如果企业微信的二次验证弹窗未弹出，请检查手机通知权限。若手机遗失，请立即拨打IT服务热线（分机号：8000）申请挂失并重置MFA令牌。
